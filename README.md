# LIFE OS — личный дневник жизни

Персональная «операционная система жизни». Ты наговариваешь голосом или пишешь
текстом в Telegram → AI распознаёт, раскладывает по категориям, тегам, настроению,
выделяет задачи и инсайты → красивый веб-дашборд показывает дневник, аналитику,
«что заметил AI» и отвечает на вопросы о твоей жизни.

**Сайт:** [mylifebookai.vercel.app](https://mylifebookai.vercel.app)

---

## 🔐 Приватность и данные (главное)

Этот репозиторий **открыт специально**, чтобы любой мог проверить, что именно
происходит с данными — а не верить на слово.

- **Дневник видит только владелец** — доступ по личной ссылке (magic-link), опционально под PIN-кодом.
- **Команда не читает записи** — внутренняя аналитика только обезличенная (счётчики, темы в %), без текста. См. `src/app/admin/`.
- **AI обрабатывает текст только для пользователя** — для резюме, ответов и аналитики. По условиям API Anthropic и OpenAI данные **не используются для обучения** их моделей.
- **Защита базы (RLS)** — прямой доступ к данным извне закрыт; ходит только серверный ключ. См. `supabase/rls.sql`.
- **Экспорт** — пользователь может скачать **все свои данные** одним файлом в любой момент (`/api/export`).
- **Удаление** — стереть аккаунт со всем содержимым в один клик (`/api/account`).
- **Шифрование** — данные шифруются в покое (Supabase) и передаются по TLS.

**Нашли уязвимость?** Сообщите приватно — см. [SECURITY.md](SECURITY.md) (`0636388813@ukr.net`). Не раскрывайте публично, пока не починим.

Полная политика — на странице `/privacy`, карта архитектуры — в `/admin/architecture`.

## 🏠 Self-host

Проект можно развернуть на своих серверах со своими ключами:
1. Создать проект в Supabase, выполнить `supabase/*.sql`.
2. Завести ключи: `TELEGRAM_BOT_TOKEN`, `NEXT_PUBLIC_SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`, `ANTHROPIC_API_KEY`, `OPENAI_API_KEY`, `TELEGRAM_WEBHOOK_SECRET`.
3. Задеплоить на Vercel (или любой Node-хостинг).
4. Настроить вебхук бота: `GET /api/setup-webhook`, команды: `GET /api/setup-commands`.
