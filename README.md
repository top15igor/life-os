# LIFE OS — личный дневник жизни

Личная «операционная система жизни»: записи голосом/текстом через Telegram →
AI раскладывает по категориям, тегам, настроению → красивый веб-дашборд с
дневником, аналитикой и поиском.

**Стек:** Next.js (React) + Supabase (Postgres) + Telegram + Whisper (голос) + Claude (анализ).
Хостинг: Vercel. Дизайн утверждён в `LIFE_OS_UI_WIREFRAMES.md`.

## Твои действия (по шагам, делаем вместе)

### Шаг 1 — создать таблицы в Supabase  ← мы здесь
1. Открой свой проект на supabase.com
2. Слева: **SQL Editor → New query**
3. Открой файл `supabase/schema.sql`, скопируй всё и вставь в редактор
4. Нажми **Run** (или Cmd+Enter)
5. Должно появиться «Success». Слева в **Table Editor** появятся таблицы (entries, categories…)

### Шаг 2 — ключи в `.env.local`  (сделаем дальше)
Скопировать `.env.example` → `.env.local` и вписать ключи.
