# LIFE OS — Социальный слой: дорожная карта

> Источник: ТЗ «LIFE OS Social» (PRD от 2026-06-27). Этот файл — рабочий план, а не пересказ ТЗ.

## Главный принцип
**Соцсеть путей, а не постов.** Делишься не картинкой, а прогрессом и опытом.
**Не пересборка — достройка.** Фундамент уже есть (Фаза 0).
**Лесенка «сначала один, потом многие»:** каждый ранний шаг ценен при НУЛЕ подписчиков и сам приводит людей.

## Главный риск и как решаем — COLD-START
Лента / подписки / сообщества **мертвы без массы людей**. Поэтому:
- **Фазы 0–2 одиночно-ценны** и шерятся во ВНЕШНИЕ сети (Instagram/Telegram/WhatsApp) → рост без внутренней плотности.
- **Соцграф (Фаза 3) — только когда есть база авторов.** Не строим пустую комнату.
- ⚠️ «MVP» из §18 ТЗ на деле растянут на Фазы 1–3 (там и лента, и подписки, и реакции — это density-фичи). Это подтверждает: одной сборкой нельзя.

---

## Фаза 0 — Фундамент ✅ ГОТОВО
- Публичный профиль `/p/<slug>` (opt-in, дефолт ВЫКЛ).
- Карточки достижений (4 типа, редактируемые: Своё/Прогресс/Мечта/Мысль) → шер во внешние сети.
- Приватность по умолчанию; реф-ссылки `/i/<код>`.
- Маппинг ТЗ: §10 профиль, §5.5 достижения, §8 основа приватности, §16 публичная книга (зачаток).

---

## Фаза 1 — Публичная книга 🎯 СЛЕДУЮЩАЯ
**Цель:** профиль превращается в «Публичную книгу» из опубликованных страниц. Одиночно-ценно — это твой публичный дневник, шеришь своей внешней аудитории.

**Строим:**
1. **Публикация записи.** На `/entry/[id]` и карточках `/diary` — кнопка «Опубликовать» + уровень приватности: 🔒 Только я · 🔗 По ссылке · 🌍 Публично. **Дефолт — приватно.**
2. **AI готовит публичную версию (§9).** Переписывает личное→публичное (тёплый, понятный текст от 1-го лица), предлагает заголовок и теги. Кнопки: Опубликовать / Оставить личным / Редактировать.
3. **AI-очистка чувствительного (§8.3).** Прячет фамилии, телефоны, адреса, мед-детали, имена детей, финансы — с предпросмотром «что именно уйдёт» перед публикацией. Финальное решение — за юзером.
4. **Публичная книга на `/p/<slug>`.** Лента опубликованных страниц (а не только цифры-агрегаты).

**БД:** на `entries` — `public` (bool) + `privacy` (own/link/public) + `public_text` (AI-версия) + `public_title`; либо отдельная `public_pages`.
**Cold-start:** 0 — шеришь книгу/страницу ссылкой наружу.
**Модерация:** пока НЕ нужна (контент твой, приватность пер-страница, AI-очистка). Маппинг ТЗ: §5.1, §8, §9, §16, §18.1–2, §18.9.

## Фаза 2 — Пути (Paths)
**Цель:** долгосрочные истории. Одиночно-ценно (твоя история), шерится наружу.

**Строим:**
1. Сущность **«Путь» (§5.3):** название, описание, старт→цель, этапы, таймлайн, прогресс.
2. Привязка публичных страниц к пути.
3. **Публичная страница пути** `/path/<id>` — красивый лонгрид-таймлайн (стиль Medium/Substack/Calm).
4. (позже в фазе) **AI собирает «главу» (§5.2)** из нескольких страниц — мини-лонгрид.

**БД:** `paths`, связь `page↔path`.
**Cold-start:** 0 — путь шерится ссылкой. Маппинг ТЗ: §5.2–5.3, §18.5.

---

## Фаза 3 — Соцграф (ТОЛЬКО при базе пользователей)
**Триггер старта:** достаточно активных публичных авторов (ориентир ≥ 200–500), иначе всё пусто.

**Строим:**
1. **Подписки (§7):** на людей, пути, темы, коллекции — НЕ только на людей (ключевое отличие).
2. **Ленты (§6):** путей / страниц / инсайтов / достижений / историй — осмысленные, не бесконечный поток.
3. **Реакции без токсичности (§12):** Вдохновляет · Полезно · Поддерживаю · Сохранил себе · Прохожу похожий путь · Спасибо за честность. Без публичных лайк-рейтингов.
4. **Импорт чужого опыта в свою LIFE OS (§15):** «Сохранить в книгу / Создать цель / Создать привычку / Подписаться на путь». ← превращает соцсеть в источник практики.
5. **Сообщества вокруг путей (§11):** АИП, Бег, Стартап, Родительство…
6. **Модерация (§22):** жалобы, скрытие, блок, модерация мед-утверждений, предупреждения о чувствительном. **Появляется ЗДЕСЬ** — когда контент виден чужим в масштабе.
7. **Коллекции (§5.4), AI-Stories, Life Wrapped, челленджи (§19).**

Маппинг ТЗ: §6, §7, §11, §12, §15, §22, §19, §18.4/6/7/8/10.

## Сквозные принципы (на всех фазах)
- **Приватность по умолчанию.** Каждая запись приватна; публикация — только вручную, с предпросмотром.
- **Анти-токсичность (§13).** Реакции = опыт, не гонка. Нет публичных счётчиков-рейтингов. Достижения = «я строю свою жизнь», не соревнование.
- **AI — помощник публикации, а не автопостинг.** Предлагает, юзер решает.

## Чего НЕ делаем рано
Лента, подписки, сообщества, инфраструктура модерации — до Фазы 3. Полный 24-секционный PRD из §23 — не пишем (двигает релиз, а не документ).
